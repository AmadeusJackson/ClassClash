# ClassClash

ClassClash is a competitive study tracking web app designed to turn academic focus into friendly rivalry.  
It allows students to track study time by class, visualize weekly progress, and eventually compete on leaderboards.

---

## 🚀 Vision

ClassClash transforms traditional study tracking into a competitive, motivating experience.

Instead of simply logging hours, students:
- Track time per class
- Compare weekly totals
- Compete on leaderboards
- Build consistent study habits

Future versions will support real user accounts, live leaderboards, and cross-class competition.

---

## 🧱 Current Features

- 📚 Class-based study timers
- ⏱ Real-time session tracking
- 📊 Weekly total aggregation
- 🏆 Top class overview
- 🎛 Settings page with profile form
- 🌐 Supabase backend integration (in progress)

---

## 🛠 Tech Stack

Frontend:
- HTML5
- CSS3
- Vanilla JavaScript (SPA-style rendering)

Backend:
- Supabase (PostgreSQL database)
- Row Level Security (RLS) enabled
- Supabase JS client
